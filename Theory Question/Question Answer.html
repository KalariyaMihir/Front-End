Question 1 - What is URL

URL - stands for Uniform Recourse Locator. 
                   - it shows the web addresses of websites.
                   - URLs occurs most commonly to refer web pages but it also used for File transfer(FTP),emails.
                   - ( https://en.wikipedia.org/wiki/URL ) this is the example of URl.
                   - in Given URL "https" is Protocol,"en" is sub domain,"wikipedia" is root domain, "org" is top domain.

                
                   there is two types of URL : 1.) Absolute 
                                               2.) Relative
                                               
                   Absolute Path : This path shows the landing page/ home page of site.
                   Relative Path : This path shows the other pages of sites , Like about, menu, contact etc...

Question 9 - HTTP and HTTPS

                -> HTTP :
                        
                    - HTTP stands for Hyper Text Transfer Protocol. 
                    - HTTP is Less Secure than HTTPS.
                    - HTTP shows the full path of url.

                -> HTTPS : 

                    - HTTPS stands for Hyper Text Transfer Protocol Secure.
                    - HTTPS is Secure.
                    - HTTPS doesn't show the full path of url it shows the encrypted path.
